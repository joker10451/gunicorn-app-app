# Стратегия трафика для LeadGid (РФ, финансовые офферы)

Цель: привести целевой трафик на лендинг `gunicorn-app-app.vercel.app`,
чтобы заявки уходили в Универсальную заявку LeadGid и конвертировались в оплату.

## Важные ограничения LeadGid
- Нужен **валидный источник трафика** (не фейк, не боты)
- Обязательно **согласие на обработку ПД** (у нас уже есть в форме)
- Дубли по телефону блокируются (406) — чистим базы
- Качество трафика влияет на апрув офферов

## Каналы по приоритету

### 1. Яндекс.Директ (платно, самый тёплый) ⭐
- Поиск по «займ онлайн», «кредит наличными», «деньги в долг»
- Гео: РФ, регионы с высоким апрувом
- Бюджет от 300-500₽/день на тест
- Посадочная: наш лендинг (UTM: utm_source=yandex, utm_campaign=search)
- Ожидание: 1 заявка ≈ 50-150₽ (зависит от региона)

### 2. VK Реклама (платно, дешевле)
- Таргет: интересы «кредиты», «займы», «финансы», возраст 21-55
- Ретаргетинг по посетителям сайта
- Бюджет от 200₽/день
- UTM: utm_source=vk, utm_campaign=target

### 3. Тизерные сети (MGID, PropellerAds) — дешево, ниже качество
- Нативные текстово-графические блоки
- Бюджет от 100₽/день
- Важно: прелендинг с проверкой возраста (21+)
- UTM: utm_source=mgid, utm_campaign=teaser

### 4. SEO-лендинг (бесплатно, долго)
- Статьи под запросы «как получить займ», «займ без отказа»
- Ссылки с форумами, отзовиками (честно, без спама)
- Результат через 2-3 месяца

### 5. Свой Telegram-канал / блог (бесплатно)
- Полезный контент про финансы → ссылка на заявку
- Конверсия выше (теплота аудитории)

## План на первую неделю (бесплатно/минимум)
1. Подключить Яндекс.Метрику + GA (сделано в коде — нужны ID)
2. Настроить UTM на все кампании (сделано — subid1-9)
3. Запустить Директ на 300₽/день, гео: 5-10 регионов
4. Через 3 дня посмотреть в Метрике: CTR, конверсию в заявку
5. Отключить неэффективные регионы/объявления

## Метрики успеха
- Заявка на сайте → цель `lead_sent` в Метрике
- Сопоставить с статусами в LeadGid (accepted/rejected)
- CR (conversion rate) сайта: норма 5-15% для фин.ниши
- CPA (cost per action): следить, чтобы < выплаты оффера
